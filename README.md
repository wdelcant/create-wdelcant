# Create Wdelcant

Crea base de aplicaciones.

## Usage

```bash
# Interactive mode
pnpm create wdelcant

# Override template
pnpm create wdelcant --template <template>

# Override name
pnpm create wdelcant --name <name>

# Get Command help
pnpm create wdelcant --help
```

## Installation

```bash
pnpm install -g create-wdelcant
```
